はたらくだけじゃだめですか？

December 31, 2025

ダメに決まってんだろ！

2025年を振り返っていきます。

学業#

3月に修了し修士（工学）をゲットした。情報系の修士課程なんだから一目でそれがわかる学位がよかった〜という日本の大学あるあるは置いておくとして、全体を総括すると「たくさん失敗してたくさん学んだ」という感想になる。

正直研究はあまりうまくいかなかった。それ自体は悲しいことだが、その過程で色々な失敗ができたとポジティブに捉えている。 成功より失敗、結果より過程から学ぶことの方が多いとはよく言われるが、それを実感した二年間だった。

修士では「どう説得力のあるストーリーを書くか」ということに最も苦しんだが、これは今も奮闘し続けている。 ストーリーというのは少し論文的すぎる気もするが、仕事も提案の連続でその点では研究と似ていると感じる。

話を2025年に戻すと、自分の中途半端な研究を引き継いだ後輩の論文が国際会議に通って良かった。自分の残したものが負債になるのをとても恐れていたので一安心した。

仕事#

18年間の学生生活を終えてついに社会人一年生になった。が、働いているという感覚はあまりなく、高校ぶりに部活に入部した気分である。

謎のIT企業入社#

業務システムとその業務システムを作る基盤を提供する会社に就職し、基盤を作るチームでソフトウェアエンジニアとして働き始めた。

まだかなり小さい企業で2025年終盤にシリーズA完了のリリースが出たところだ 1。 知名度がないので文系の友人（いない）に説明する時は次世代の SAP を作っていると言い、エンジニアに説明する時は Firebase/Supabase のような BaaS や Cloudflare のような PaaS と似た何かを作っていると適当に話している。

会社の見解とは関係ない個人の野望だが、業務システムの文脈では Shopify 業務システム開発基盤の文脈では Vercel のような企業にしたいと思っている 2。 開発者のためのソフトウェア開発であることや良いものを作っているという自負から、10回就活をやり直しても10回今の企業に入ると思うレベルで大満足である。

自分がこの会社でやりたいことは技術力を高めるとか高い給料を貰うとかではなく、プロダクトを当てるということなので、良い仕事をしてその答え合わせを30代でしたい。

Code review > Pull request#

フルタイム社会人エンジニアになって一番の変化は日頃からコードレビューをするようになったことである。GitHubのプロフィールを見ると Code review > Pull requestとなっており、これは当然初めてのことだ。

レビュワーはランダムかつ平等にアサインされる仕組み 3 なのだが、同僚と比べて何倍も時間がかかるので労働に占めるレビューの割合は必然的に高くなった。 Web UI上では限界を感じ、手元にチェックアウトしてレビューすることも多かった 4。

コードはAIが書くことも増えたが、今のところ人間のレビューの価値は高いままだと感じるので引き続き頑張っていきたい。

生活#

要改善事項

Stay home#

今年は人生で最も家にいた時間が長かった年だった。コロナ越えである。

会社は3ヶ月に1回オンサイトイベントという名の同僚と美味しいご飯を食べられる日があるのみで、フルリモートかつ裁量労働制である。 そもそもエンジニアは東京に全然いないのでそれ以外でオフィスに行こうという気にはならなかった。引きこもり耐性◎だったため1ヶ月ゴミ捨て以外で外出無しということもあったと思う。

個人的に今の労働環境は最高なのだが、人と接しなさすぎることによる弊害がちょくちょく出てきている。 今年は2回風邪を引いたが、どちらもオンサイトイベントの直後だったので明らかに免疫力が低下している感がありよくない 5。

健康#

流石に危機感を覚えたので近所のスポーツジムを契約した。極端な人間なので一時期は週4~5で通っていた。 睡眠は改善したし大病の予防になっている感はあるが、結局人と接することは皆無なので風邪の問題は解消されている気がしない。

体重をキープすることも課題である。毎年食べる量が減っていて普通にまずい。特に今年は氷食症が悪化して夏の間は3ヶ月間くらいお腹を壊していた。 健康は2026年のテーマである。

旅行#

修論を提出してからイタリア・オーストリアに行った。

ドイツの大学院に通っている友達が色々手配してくれたので超快適だった。 美術館がとても楽しかったのと、フィレンツェの街並みがとても気にいった。

他にもオペラの立ち見&断食がきつかったり、友達がぼったくられたり、時差ボケで死にかけたりして楽しかった 6。 食べ物は帰国前にフィウミチーノという空港近くの街に泊まった時に行った地元レストランがダントツで美味しかった 7。海沿いかつ海鮮の優勝。

最後に#

修論を提出して旅行してコードを読んでいたら1年が終わっていた。

今の仕事は多分外資系のスタートアップに多いタイプの仕事で、なんとなく30歳くらいで転職してやりたいと思っていたことが運良く数年ショートカットしてやれている。
拾ってくれた同僚に感謝しつつ来年も頑張ろうと思う。

皆んな来年も仲良くしてネ 8
